# Airline Logos

Full-color airline logo assets used by Hubyt flight cards.

Files are named by IATA carrier code, for example `DL.png` and `WN.png`.
The app fetches these from:

`https://raw.githubusercontent.com/bptworld/hubyt-cards/main/assets/airlines/{IATA}.png`

See `manifest.json` for source URLs and notes. Airline names, logos, and
trademarks remain owned by their respective holders.
